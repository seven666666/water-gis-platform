# 水利GIS平台 (water-gis-platform)

## 项目概述
- 目标：基于 Cesium + Three.js 的水库河流监测可视化系统
- 技术栈：Vue 3 + Vite + Cesium + Three.js + ECharts

## 功能模块
1. 地图基础展示
2. 水库/河流数据可视化
3. 实时监测数据展示
4. 三维动态效果
